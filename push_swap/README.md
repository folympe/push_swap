# Push_swap

## Description
Push Swap est un programme qui trie une liste d'entiers.
Il utilise deux piles : a et b.
Les nombres sont initialement dans la pile a.
Le tri doit être effectué avec un nombre limité d'instructions.
Les opérations incluent swap, push, rotate et reverse rotate.
Chaque action doit être optimisée pour réduire le nombre de coups.
Le programme affiche la liste des instructions pour trier les données.
Différentes stratégies sont utilisées selon la taille de la liste.
L'objectif est d'obtenir un tri rapide et efficace.
Ce projet met l'accent sur l'optimisation et la logique algorithmique.

# Instructions
cree une initilisation avec des algo pour differents taille de liste avec des outils comme ;
| Command | Description |
| --- | --- |
| **sa / sb / ss** | Swap the first 2 elements at the top of stack A / B / both. |
| **pa / pb** | Push the first element from one stack to the top of the other. |
| **ra / rb / rr** | Shift up all elements of a stack by 1 (the first becomes the last). |
| **rra / rrb / rrr** | Shift down all elements by 1 (the last becomes the first). |
et savoir bouger les nombres melanger dans les deux lists pour les remettres en place
